Пример задачи для применения паттерна Command

Текстовый редактор, в котором могут выполняться команды: вставка текста, удаление текста,
изменение текста. Каждое действие можно отменить (Undo) или повторить (Redo).

Каждое действие (команда) будет инкапсулироваться в отдельный объект;
Команды сохраняются в истории для отиены и повторения;
Позволяется добавлять действия;

Command — абстрактный класс с методами undo() и execute();
Наследуют комманды — Insert_, Delete_, Replace_;
Editor — хранит текст и предоставляет методы для работы с ним.
Manager — управляет историей команд (для Undo и Redo).
